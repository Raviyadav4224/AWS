1. SQS -

    | Feature            | Standard Queue   | FIFO Queue        |
    | ------------------ | ---------------- | ----------------- |
    | Max message size   | 256 KB           | 256 KB            |
    | Message retention  | 1 min – 14 days  | Same              |
    | Visibility timeout | 0 sec – 12 hrs   | Same              |
    | Delay queue        | 0 sec – 15 min   | Same              |
    | Throughput         | Nearly unlimited | Lower             |
    | Ordering           | Best effort      | Guaranteed        |
    | Deduplication      | No               | Yes               |
    | Long polling       | Up to 20 sec     | Same              |

2. AWS KMS -

    * Max direct encrypt payload = 4KB

3. Lambda -

    | Limit              | Value          |
    | ------------------ | -------------- |
    | Lambda RAM         | 128 MB - 10 GB |
    | ZIP upload size    | 50 MB          |
    | Unzipped package   | 250 MB         |
    | Container image    | 10 GB          |
    | Total code storage | 75 GB default  |
    |  Timeout           | 1-15sec, default 3sec |

4. ECS -

    * etc/ecs/ecs.config - (For EC2 Launch Type)

      ECS_CLUSTER=my-cluster (Associates EC2 with the Cluster)
      ECS_ENABLE_TASK_IAM_ROLE=true (Task Role used by the Application to access different services)
      ECS_AVAILABLE_LOGGING_DRIVERS=["json-file","awslogs"]

5. Elastic Beanstalk -

    Application Version Size = 512 MB
    Application Versions = 1000 per Region, If no of versions exceeds deployment fails

6. XRAY -

    * Environment variables to communicate with X-Ray
      _X_AMZN_TRACE_ID: contains the tracing header
      AWS_XRAY_CONTEXT_MISSING: by default, LOG_ERROR
      AWS_XRAY_DAEMON_ADDRESS: the X-Ray Daemon IP_ADDRESS:PORT

7. DynamoDB -

    RCU  = 4 KB
    WCU  = 1 KB
    Item = 400 KB
    Query Result = 1 MB, if exceeds then first 1MB + LastEvaluated Key
    Transaction = 100 items, Maximum 4 MB
    1 Eventually Consistent Read consumes Half RCU compared to Strongly Consistent Read 

8. CloudWatch -

    Annotations - 50 at max per trace, user added data and INDEXED
    Metrics - Standard - 1min = 60sec, High Resolution = Can have 1s / 15s / 30s - Higher cost
